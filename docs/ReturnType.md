# ReturnType

Тип заказа для фильтрации:  * `RETURN` — возврат.  * `UNREDEEMED` — невыкуп.  Если не указывать, в ответе будут и возвраты, и невыкупы. 

## Enum

* `UNREDEEMED` (value: `'UNREDEEMED'`)

* `RETURN` (value: `'RETURN'`)
