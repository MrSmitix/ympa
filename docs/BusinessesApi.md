# **GetBusinessSettings**
> GetBusinessSettingsResponse GetBusinessSettings (long businessId)

Настройки кабинета

Возвращает информацию о настройках кабинета, идентификатор которого указан в запросе. |**⚙️ Лимит:** 1000 запросов в час| |-|
