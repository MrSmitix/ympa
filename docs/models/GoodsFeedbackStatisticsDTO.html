GoodsFeedbackStatisticsDTO

  • rating : Integer
    Оценка товара.
  • commentsCount : Long
    Количество комментариев к отзыву. Учитываются только ответы на отзывы, а не дочерние комментарии.
  • recommended : Boolean
    Рекомендуют ли этот товар.
  • paidAmount : Long
    Количество баллов Плюса, которое автор получил за отзыв.
