GetChatInfoDTO

  • chatId : Long
    Идентификатор чата.
  • orderId : Long
    Идентификатор заказа.
  • type : ChatType
  • status : ChatStatusType
  • createdAt : Date
    Дата и время создания чата. Формат даты: ISO 8601 со смещением относительно UTC. Например, `2017-11-21T00:00:00+03:00`.
  • updatedAt : Date
    Дата и время последнего сообщения в чате.
