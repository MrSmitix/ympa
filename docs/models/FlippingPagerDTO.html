FlippingPagerDTO

  • total : Integer
    Сколько всего найдено элементов.
  • from : Integer
    Начальный номер найденного элемента на странице.
  • to : Integer
    Конечный номер найденного элемента на странице.
  • currentPage : Integer
    Текущая страница.
  • pagesCount : Integer
    Общее количество страниц.
  • pageSize : Integer
    Размер страницы.
