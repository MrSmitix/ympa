# OrderUpdateStatusType

Изменился ли статус заказа:  * `OK` — статус изменен.  * `ERROR` — статус не изменен. В этом случае появится сообщение об ошибке в параметре `errorDetails`. 

## Enum

* `OK` (value: `'OK'`)

* `ERROR` (value: `'ERROR'`)
